# GWAS_formatting_toolkit
This repository contains a collection of scripts for formatting soybean phenotype data to match with genotype data.

The toolkit is designed to simplify preprocessing workflows.

### Compatible genotype
Soy1066

Soy775
#### Source: https://soykb.org/public_data.php
### Compatible phenotype
Soybean colection from GRIN
#### Source: https://www.grin-global.org/

## Requirements
Python
Common libraries (pandas, numpy, matplotlib)
